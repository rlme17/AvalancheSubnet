# RichardToken

The RichardToken is an ERC20 token that can be minted and burned by the owner. It has a total supply of 1,000,000 tokens. Tokens can be transferred between accounts, but transfers to the zero address are not allowed.

### Usage

To use the RichardToken, you will need to first deploy it to the Ethereum blockchain. Once it is deployed, you can interact with it using the following functions:

* `mint`: This function mints new RichardTokens. Only the owner of the contract can mint new tokens.
* `burn`: This function burns RichardTokens. Only the owner of the contract can burn tokens.
* `transfer`: This function transfers RichardTokens from one account to another. Anyone can call this function.
